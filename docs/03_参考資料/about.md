# NULL-N PractiCase について(Authorship)

## 著作・運営

NULL-N PractiCase は、**NULL-N が企画、設計方針、教材方針、要件判断、品質判断、公開判断を
行っている**エンジニア育成教材です。

実装や文書作成の一部では AI ツールを補助的に利用していますが、本プロジェクトの方向性、学習設計、
採否判断、品質基準、リポジトリの運営方針は NULL-N が決定しています。

## Authorship (English)

NULL-N PractiCase is planned, directed, reviewed, and maintained by **NULL-N**.

The concept, learning design, requirements, acceptance decisions, quality standards,
repository operation policy, and release decisions are owned by NULL-N.

AI tools were used as implementation and drafting assistants, but the project direction,
instructional design, and final decisions are made by NULL-N.

## Contribution Policy

This repository is maintained by NULL-N.

At this stage, external pull requests are not part of the official workflow.
If issues are enabled, they are used for feedback and reports only.
Changes are reviewed, selected, and implemented by NULL-N.

外部からの Pull Request は、現時点では公式の運用に含めていません。
Issue(有効にしている場合)はフィードバックと不具合報告の窓口で、修正の判断と実装は NULL-N が行います。
