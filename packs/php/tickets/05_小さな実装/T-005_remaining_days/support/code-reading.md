# T-005 コードリーディング(新機能でも、書く前に読む)

今回は「壊れたコードを読む」課題ではありません。それでも読むことから始めます —
**新しいコードは、周りの既存コードに馴染んで初めて「良い追加」になる**からです(実装の答えは書いていません)。
読むのは3つ: ①これから使う道具 ②繋ぎ先の画面 ③合格条件のテスト。

## ① 使う道具を読む: src/Support/Clock.php

label() の基準日はここから取ります。短いクラスなので全部読めます。

```php
public static function now(): \DateTimeImmutable
{
    return self::$fixed ?? new \DateTimeImmutable('now', new \DateTimeZone(self::TIMEZONE));
}

public static function today(): \DateTimeImmutable
{
    return self::now()->setTime(0, 0, 0);
}
```

1行ずつ:

| コード | 何をしているか |
|---|---|
| `self::$fixed ?? new ...('now')` | 固定日時がセットされていればそれを、無ければ本物の現在時刻を返す(`??` は null合体) |
| `new \DateTimeZone(self::TIMEZONE)` | タイムゾーンを東京に明示(サーバ設定に依存させない) |
| `today()` = `now()->setTime(0, 0, 0)` | **時刻を 0:00:00 に丸めた「日付だけ」**を返す。「時刻は考慮しない」の土台 |
| `fix()` / `clear()` | テスト専用。基準日を固定 → 実行日が変わってもテスト結果が揺れない(ARC-5 の目的そのもの) |

**読み取れる型**: 「現在日時の取得口を1つに絞る」から、テストで時間を止められる。
あなたの label() が `date()` を直接呼んだら、この仕組みの外に出てしまいます(それが ARC-5 違反)。

## ② 繋ぎ先を読む: public/projects/show.php

締切を表示している行はここです。

```php
<tr><th>応募締切</th><td><?= e($project['deadline']) ?></td></tr>
```

| コード | 何をしているか |
|---|---|
| `<?= ... ?>` | `<?php echo ...; ?>` の短縮形。値を HTML に出す |
| `e(...)` | このアプリ共通のエスケープ関数(SEC-2: 画面に出す値は必ず通す)。追記分も同じ扱いにする |
| `$project['deadline']` | DB から来た `YYYY-MM-DD` の文字列。label() にそのまま渡せる形 |

この行に「(残りN日)」を**追記するだけ**にします。日数の計算をこのファイルに書き始めたら、
それは ARC-1(画面にロジックを書かない)からの逸脱です — 計算は新設クラスの仕事。
なお、show.php で新しいクラスを使うには、ファイル上部の `use` の並びに1行足す必要があります(既存の並びが手本)。

## ③ 合格条件を読む: tests/checks/T-005.php

テストは「仕様の表をコードに翻訳したもの」です。読み方:

| コード | 何をしているか |
|---|---|
| `Clock::fix('2026-07-01 10:00:00')` | 基準日を固定。以降 `Clock::today()` は 2026-07-01 の 0:00 を返す |
| `assertSame('残り9日', ...label('2026-07-10'))` | 期待値と実際の値が**完全一致**か(文言も仕様のうち) |
| `Clock::fix('2026-07-01 23:00:00')` の側 | **23時に実行しても「本日締切」** — 「時刻は考慮しない」をここで検査 |
| `Clock::clear()` | 固定を解除。テストが終わったら時間を元に戻す(後始末) |

## ここから自分で

3つ読んだら、材料は揃っています。仕様の表(spec.md)を if の言葉に翻訳し、
Clock.php と同じ「クラスの書き出しの型」で `RemainingDays` を作ってください。
2つの日付の差は `DateTimeImmutable` の `diff()` — 戻り値から日数をどう取り出すかは、
PHP マニュアルの `DateInterval::format` を調べます(調べて使うのも実務の型です)。
