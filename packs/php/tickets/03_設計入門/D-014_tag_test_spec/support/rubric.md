- [ ] タグの登録(S-12)・絞り込み(S-03)・表示(S-04)、それぞれのケースがある

- [ ] すべてのケースに期待結果が書かれている
- [ ] 「実施結果」「OK/NG」は書いていない(このチケットは設計のみ・実行はしない)
