# T-002 セルフチェック(rubric)

## 調査と修正

- [ ] 画面 → パラメータ → Repository → SQL の経路を実際に追ったか(当てずっぽうで直していないか)
- [ ] 原因を「どの層の・何が」の形で言葉にできるか
- [ ] ON のときだけでなく **OFF のとき絞り込まれないこと**も確認したか(仕様の両側)

## テスト

- [ ] ON / OFF 両方の挙動をテストで固定したか
- [ ] check T-002 と共通テストの両方が通ったか

## 提出

- [ ] scope 外を変更していないか(CHG-1)
- [ ] PR 本文に原因・影響範囲(他の検索条件に影響しない根拠)を書いたか
- [ ] 振り返りを書き、Redmineへ結果をコメントして`Closed`にしたか
