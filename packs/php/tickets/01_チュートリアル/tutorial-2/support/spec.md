## countByTag() のルール

| 入力 | 出力 |
|---|---|
| 複数の案件にまたがるタグ | 出てきた回数だけ数える(例: PHP が2案件 → `PHP: 2`) |
| 同じ案件内で重複するタグ | それも回数として数える |
| 案件が0件 | 空の配列 `[]` |
| タグが空の案件 | 集計に影響しない(スキップされるのと同じ結果) |

- キーはタグ名(文字列)、値は件数(整数)
- 並び順は「最初に登場した順」で自然に決まる(foreach で素直に書けばそうなる)

## 参照

- 書き方の指定(foreach 二重)はチケット本文
- 使ってよい道具: PHP の標準機能のみ。新しいクラスやファイルは作らない(TODO の場所を埋めるだけ)
