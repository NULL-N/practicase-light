# D-025 セルフチェック(rubric)

- [ ] 部署名・氏名・メール・状態・部署への参照をNOT NULLにしたか
- [ ] 状態にDEFAULT(在籍中相当)の初期値を検討したか
- [ ] 作成日時のようなカラムがあるなら、DEFAULT(登録時点を自動記録)を検討したか
- [ ] NULL可にしたカラムがあれば、「なぜ空でもよいか」を業務の言葉で説明したか
      (「なんとなく」ではなく「在籍中は発生しないから」のような理由)
- [ ] UNIQUE・重複禁止・キーの話が主題になっていないか(それは次段階以降の話)
- [ ] 給与・勤怠・評価・個人番号・住所・健康情報のいずれにも触れていないか
