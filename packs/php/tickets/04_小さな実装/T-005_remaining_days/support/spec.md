# T-005 仕様(残り応募日数の表示)

対象画面: S-04 案件詳細。応募締切の欄を「2026-07-10(残り9日)」のような形にする。

## 表示ルール

`RemainingDays::label($deadline)` が返す文言(括弧は画面側で付けてよい):

| 締切日と基準日(本日)の関係 | 返す文言 | 例(本日 = 2026-07-01) |
|---|---|---|
| 締切が本日より後 | `残り{N}日` | 締切 2026-07-10 → 「残り9日」 |
| 締切が本日 | `本日締切` | 締切 2026-07-01 → 「本日締切」 |
| 締切が本日より前 | `締切済み` | 締切 2026-06-30 → 「締切済み」 |

- N は日付の差(暦日)。時刻は考慮しない(G-2: 締切日の 23:59:59 までは応募可、はこの表示と矛盾しない —
  「本日締切」の案件はまだ応募できる)
- どのロールで閲覧しても同じ表示とする(client / admin は締切済み案件の詳細も見られるため「締切済み」が出る)

## 基準日

- 基準日はシステム日付(実行したその日)。実装は Clock::today() を使う(ARC-5)
- 自動テストでは `Clock::fix()` で基準日を固定して検証する(実行日によって結果が変わるテストにしない)
