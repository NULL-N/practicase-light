# T-014 rubric

- [ ] `projects` テーブルだけで解けると判断できた
- [ ] `status = 'open'` で対象を絞った
- [ ] 返す列を `title`, `deadline`, `hourly_rate` に絞った
- [ ] `deadline` の昇順で並べた
- [ ] アプリ本体の PHP ファイルを変更していない

## 提出前の共通確認

- [ ] 課題のcheckがPASSし、課題固有のチェック項目をすべて確認した
- [ ] commit・push前に`git status --short`で無関係な変更がないことを確認した

## 提出後の共通確認

- [ ] Pull Requestをmergeした
- [ ] merge後に`support/debrief/`がある場合は自分の提出と突き合わせた
- [ ] 突き合わせで見つけた違いを踏まえて振り返りを書いた
- [ ] RedmineへPASS結果・Pull RequestのURL・振り返りをコメントし、チケットを`Closed`にした
