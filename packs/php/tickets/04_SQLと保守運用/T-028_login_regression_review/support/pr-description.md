# PR: T-019 admin ログイン不具合の修正

## 背景

admin ロールのユーザーがログインできない不具合(T-019)が報告されました。

## 原因

`AuthService::attempt()` に、`role === 'admin'` を弾いてしまう条件が紛れ込んでいました。

## 変更内容

- 停止判定(`status`)と admin 判定を1つの条件にまとめ、見通しをよくしました
- 修正後、admin でログインできることをテストで確認しました

## 確認方法

```text
docker compose exec app php tools/check.php T-019
```

を実行し、admin ログインのテストが通ることを確認しました。念のため、少し古い
テストケース(重複していた suspended の確認)は整理し、必要なチェックだけ残しました。

## レビューしてほしい点

特にありません。admin ログインが直っていることだけ確認してもらえれば OK です。
