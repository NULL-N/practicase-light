# T-028 仕様メモ

参照する資料:

- `support/pr-description.md`(PR の説明)
- `support/diff.patch`(実際の差分。適用はしない、読むだけ)
- `support/related-spec.md`(判断材料へのポインタ)

必要な条件:

- `reports/T-028_review.md` に、次のタグを付けて指摘を書く
  - `[bug]`(2件以上): 直っていない・新しく壊れている問題
  - `[impact]`(1件以上): その問題の実務上の影響
  - `[fix]` または「対応方針」(1件以上): どう直すべきか
- コードは変更しない(scope は空。レビューのみの課題)
